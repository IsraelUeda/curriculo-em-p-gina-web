Projeto Finplan

O FinPlan é uma aplicação web que ajuda os usuários a gerenciar suas finanças de maneira intuitiva e prática. Com uma interface simples e acessível, o FinPlan permite o cadastro de transações, categorização de gastos, visualização de relatórios e muito mais. Inspirado na experiência gamificada do Duolingo, o aplicativo também oferece desafios e recompensas para incentivar o bom gerenciamento financeiro. As técnologias usadas formação para o Front-End: HTML, CSS, JavaScript, Bootstrap 5. Back-End: Python(Flask). Banco de dados: PostgreSQL. Machine Learning: Pytorch. Autenticação: JWT, OAuth Link.
